# 🏠 LandChain

A blockchain-based land ownership verification system built on the Stellar ecosystem.

## 📌 Overview

LandChain is a decentralized platform designed to securely register, transfer, and verify land ownership records using blockchain technology. The system ensures transparency, prevents fraudulent ownership claims, and maintains an immutable ownership history.

## 🚀 Features

* 🔐 User Authentication
* 🏠 Register New Land
* 📄 View Owned Lands
* 🔄 Transfer Land Ownership
* 🔍 Verify Land by Land ID
* ⛓️ Blockchain-based Ownership Records
* 📜 Ownership History Tracking
* 👨‍💼 Admin Dashboard for Land Management

### Clone the Repository

```bash

```

### Install Dependencies

```bash
npm install
```

### Start the Development Server

```bash
npm run dev
```

## 🌐 How It Works

1. Users register and connect their Stellar wallet.
2. Admin registers land details.
3. Land ownership data is stored securely.
4. Users can transfer ownership.
5. Ownership history is permanently recorded.
6. Anyone can verify land authenticity using the Land ID.
